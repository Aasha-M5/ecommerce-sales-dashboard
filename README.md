# E-Commerce Sales Dashboard

## Project Overview
This project analyzes e-commerce sales data using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab
- Power BI

## Features
- KPI Cards
- Sales Analysis
- Profit Analysis
- Monthly Sales Trend
- Region-wise Sales Distribution
- Interactive Dashboard Filter
